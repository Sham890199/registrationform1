📋 Registration Form Project
📌 Overview

This project is a simple Registration Form created using HTML and CSS. It allows users to enter their basic details like name, email, contact number, gender, country, and state.

🚀 Features

User-friendly form layout

Input fields for personal details

Dropdown selection for gender, country, and state

Submit and Reset buttons

Basic styling using CSS

Responsive centered form design

🛠️ Technologies Used

HTML5 – for structure

CSS3 – for styling

📂 Project Structure
registration-form/
│── index.html
│── README.md
🧾 Form Fields

Name

Email

Contact Number

Gender (Dropdown)

Country (Dropdown)

State (Dropdown)

▶️ How to Run

Download or clone the project

Open the index.html file in any web browser

Fill in the form and test functionality

⚠️ Notes / Improvements

Email input should use type="email" instead of text

Contact field can use type="tel"

Add form validation using JavaScript

Improve styling (spacing, shadows, alignment)

Connect to backend for real data storage
